Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passew o mouse sobre os termos para descobrir qual tag devera utilizar em casa caso.

  • Paragrafos
  • Titulos
  • Imagens
  • Abreviacoes
  • Codigo-fonte
  • Listas numeradas
  • Listas com demarcadores
  • Enfase
  • Negrito
  • Citacoes (ficara entre aspas)
  • Citacoes completas (ficara com um paragrafo)
